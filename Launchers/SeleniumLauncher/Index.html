Selenium Launcher REST API

Controllers - bat, ie, screen

bat: Runs, stops and gets the status of the named batch process.
/bat - get, delete (all batch processes)
/bat/hub - get, post, delete
/bat/ie7 - get, post, delete
/bat/ie8 - get, post, delete
/bat/ie9 - get, post, delete

ie: Gets the status of and stops running instances of Internet Explorer.
/ie - get, delete

screen: Gets a screen shot of the type requested.
/screen/screen.png - get
/screen/screen.bmp - get
/screen/screen.jpg - get
/screen/screen.jpeg - get
